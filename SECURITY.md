# Security

Please do not report security vulnerabilities in public GitHub issues.

Email security reports to `security@apviso.com` with:

- Affected APVISO Runner version.
- Installation method, such as npm or Docker.
- Impact and reproduction steps.
- Relevant logs with all tokens, API keys, target credentials, cookies, and
  private URLs removed.

Apviso will acknowledge reports and coordinate fixes according to severity.
